# ColorTool
- **A  Color Manipulation Tool (Color Lighten and Darken)**
- **Built using HTML, CSS (BootStrap) and Vanilla JavaScript**
- **Converts Hex Color Codes to RGB Values and vice-versa**
- **Handles both 3 and 6 digit Hex Values**
